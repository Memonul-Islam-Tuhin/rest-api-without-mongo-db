# REST API 🚀

A simple REST API project built with **Node.js** and **Express.js**.  
This repository demonstrates how to structure a backend application using controllers, models, routes, and views.

---

## 📂 Project Structure


  I. REST-API/

    │── controllers/   # Business logic for handling requests
    │── models/        # Data models and schemas
    │── routes/        # API route definitions
    │── views/         # View templates (if applicable)
    │── app.js         # Express app configuration
    │── index.js       # Entry point of the application
    │── .env           # Environment variables
    │── package.json   # Dependencies and scripts
    │── yarn.lock      # Yarn lock file
